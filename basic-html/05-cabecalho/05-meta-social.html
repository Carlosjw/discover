<meta> para redes sociais.
- Conteúdo especial quando formos compartilhar o link de nossa página
- Open Graph: Facebook
- property="og:image": exibe uma imagem
- property="og:description": pega a descrição do "content=" para mostrar no compartilhamento.
- property="og:title": exibe um título especial.
- content="...": conteúdo que será exibido

- twitter
- name="twitter:title"
- content="..."
