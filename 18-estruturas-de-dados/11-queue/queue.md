# Queue

Tradução de Queue é fila.

Como uma fila d euma loja ou restaurante.

    * Linear
    * O primeira a entrar é o primeiro a sair

## Conceitos

    * FIFO: First In First Out
      * O primeiro a entrar na fila, é o primeiro a sair dela
    * Font(frente) é a referência do primeiro elemento a entrar na fila
    * Back(fundo) é a referência do último elemento a entrar na fila
